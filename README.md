# Olá, sou Felipe, seja bem vindo!

Construí minha vivência inicialmente em Engenharia de Software, com atuação em
desenvolvimento fullstack, onde pude desenvolver conhecimento avançado em arquitetura de
sistemas, integração de APIs, modelagem de dados e boas práticas de desenvolvimento. Nos
últimos anos, direcionei minha carreira de forma estruturada para a área de dados,
aprofundando conhecimentos em tecnologias como Python, SQL, Apache Spark, Databricks e
serviços AWS, além de fundamentos de Data Engineering, Analytics e Data Science.

Atuei no desenvolvimento de sistemas escaláveis, otimização de consultas e modelagem de
dados, além da automação de pipelines CI/CD, reduzindo em cerca de 30% o tempo de entrega
de soluções. Também tive contato direto com práticas de observabilidade, testes
automatizados e construção de aplicações orientadas a performance, sempre com foco em
qualidade, governança e eficiência operacional.

Atualmente, estou em especialização em Engenharia de Dados e formação em Análise e
Ciência de Dados, consolidando minha transição para atuar de forma completa no ciclo de
dados, desde a ingestão até a disponibilização para consumo analítico.
